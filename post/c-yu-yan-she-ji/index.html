练习2-14 求奇数分之一序列前N项和

输入格式:
输入在一行中给出一个正整数N。

输出格式:
在一行中按照“sum = S”的格式输出部分和的值S，精确到小数点后6位。题目保证计算结果不超过双精度范围。

输入样例:
23
输出样例:
sum = 2.549541

#include <stdio.h>
int main(){
double sum,item;
int N,denominator;
int i;
sum = 0;
denominator=1;
scanf("%d",&N);
for(i=1;i<=N;i++){
sum=sum+1.0/(denominator);
denominator=denominator+2;
}
printf("sum = %.6f",sum);
return 0;
}
